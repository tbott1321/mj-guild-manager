Required Columns

The spreadsheet layout should include these exact column headers:

  • Rank

  • Kills

The import will keep admin-only profile fields already stored in the app, such as Mana, Sigils, Alt Account, Troop Comp, Communication details, and Comments. It only updates the fields coming from the weekly spreadsheet unless the player is brand new.
